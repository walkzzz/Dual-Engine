# Dual-Engine 基准测试

## 运行基准测试

```bash
# 运行所有基准测试
cargo bench

# 运行特定基准测试
cargo bench --bench engine_bench

# 运行并生成 HTML 报告
cargo bench -- --output-format bencher | cargo-criterion
```

## 基准测试套件

### 1. 引擎延迟测试

测试引擎初始化和请求处理的延迟。

### 2. 速率限制测试

测试速率限制器的性能影响。

### 3. 验证器测试

测试输入验证的性能。

## 性能指标

| 操作 | 目标延迟 | 说明 |
|-----|---------|------|
| 引擎初始化 | < 100ms | 懒加载场景 |
| 请求处理 | < 2s | 包含 API 调用 |
| 速率限制检查 | < 1ms | 不应成为瓶颈 |
| 输入验证 | < 1ms | 简单验证 |

## 优化建议

根据基准测试结果优化：

1. 如果引擎初始化慢 → 检查懒加载实现
2. 如果速率限制慢 → 考虑无锁实现
3. 如果验证慢 → 减少正则表达式使用
